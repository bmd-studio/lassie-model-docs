# verify_module_access( string $person_id, string $module_id, string $section_name = FALSE, $log = TRUE )

Verify the access of a specific person for a specific method in the module. This will direct a request to our person rights table which will check if the record exists. And if so has the required rights.

Verify the access of a specific person for a specific method in the module. This will direct a request to our person rights table which will check if the record exists. And if so has the required rights.

Furthermore, note that the combination of user_id, module_name and section_name must be unique within the MYSQL table structure. This to prevent multiple entries of the same right. CREATE UNIQUE INDEX person_access ON rights_person (group_id, person_id); CREATE UNIQUE INDEX user_access ON rights_group (group_id, module_name, section_name); (SOURCE: http://stackoverflow.com/questions/5955821/mysql-make-a-combination-of-columns-unique)
